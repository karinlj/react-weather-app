# Weather App

## Description

Weather App with React.js and Openweathermap API

## Installation

To install

`npm install`

To run

`npm start`

`npm run server`

## To build

`npm run build`

Then use the dist folder you get.

## Built With

- React.js
- Scss
- Openweathermap API
